# Sistem Peminjaman Ruangan PENS (Frontend)

Aplikasi web frontend untuk sistem peminjaman ruangan kampus, dibangun menggunakan **React** dan **TypeScript**.

## Fitur Utama
- **User Dashboard:** Peminjaman ruangan, riwayat, dan pembatalan mandiri.
- **Admin Dashboard:** Manajemen ruangan, user, dan persetujuan peminjaman.
- **Responsive Design:** Tampilan optimal hanya di Desktop

## Tech Stack
- React
- TypeScript
- Vite
- Axios (API Consumption)
- CSS Modules

## Cara Menjalankan
1. Clone repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy .env.example menjadi .env dan sesuaikan url
4. Jalankan server development:
  ```bash
   npm run dev
  ```
