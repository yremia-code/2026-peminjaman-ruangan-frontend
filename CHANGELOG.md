# Changelog

Semua perubahan penting pada proyek ini akan didokumentasikan dalam file ini.

## [1.0.0] - 2026-02-14

### Added
- **Dashboard User:** Halaman utama user dengan layout responsif dan sticky sidebar.
- **Booking System:** Form peminjaman ruangan dengan validasi tanggal.
- **Riwayat Peminjaman:** Fitur melihat status peminjaman (Pending, Approved, Rejected, Canceled).
- **Self-Cancel:** User dapat membatalkan peminjaman yang masih berstatus Pending.
- **Search & Filter:** Pencarian realtime untuk ruangan dan riwayat peminjaman.
- **Auth Integration:** Login page terintegrasi dengan JWT Backend.
- **Identitas:** Update Favicon dan Title sesuai branding PENS.

### Fixed
- **UI/UX:** Perbaikan masalah scrolling pada dashboard user (sidebar dan konten independen).
- **Accessibility:** Penambahan atribut aria pada form input.
